How to use font awesome icons:


Include in the <head> of your html document:


<script src="https://kit.fontawesome.com/ea11f319af.js" crossorigin="anonymous"></script>


When you want to use an icon, use the following html:


<i class="fas fa-iconName""></i>

where "iconName" is the name of the icon.



For example, for a dog icon:


dog


You would use the following:


<i class="fas fa-dog""></i>
